# High-End Agency Website Prompt (Valiente-Inspired + Custom Brand System)

---

## 1. OVERALL DIRECTION

Build a modern, high-end branding studio / creative agency website inspired by premium editorial and experimental agency sites like Valiente Brands.

The website should feel:
- Bold, typography-led, and editorial
- Scroll-driven storytelling experience
- Minimal UI with strong visual hierarchy
- Cinematic transitions and smooth motion
- Manifesto-style copywriting
- Premium creative studio aesthetic

---

## 2. COLOR SYSTEM (STRICT)

Use ONLY the following palette:

- Background: #ffe8c2 (warm paper/editorial canvas)
- Primary: #9e6240 (clay / CTA / highlights)
- Secondary: #659157 (muted green / accents)
- Font Primary: #221e22 (main text / ink)
- Font Secondary: #7f6a93 (muted metadata)

### Rules:
- Background always #ffe8c2
- Headings: #221e22
- Body: #221e22 (lower opacity allowed)
- CTAs: #9e6240
- Hover/links: #659157
- Metadata: #7f6a93
- No extra colors allowed

---

## 3. TYPOGRAPHY

- Large editorial-style headings (grotesk or serif-sans hybrid)
- Tight, intentional spacing
- Strong hierarchy:
  - H1: oversized hero text
  - H2: section headers
  - Body: clean and readable
  - Metadata: muted secondary tone

---

## 4. DESIGN PRINCIPLES

- Minimal but expressive layout
- Strong grid alignment with whitespace
- Scroll-based narrative sections
- Subtle grain/noise texture overlay
- Smooth fade/translate animations
- Hover interactions (underline, scale, color shift)

---

## 5. SITE STRUCTURE

### HERO SECTION
- Bold headline (e.g., “We design brands with clarity and conviction.”)
- Short manifesto subtext
- CTA buttons:
  - Primary (filled #9e6240)
  - Secondary (outline)

Effects:
- Fade-in on load
- Slight scroll parallax

---

### MANIFESTO SECTION
- Short editorial paragraphs
- Tone: confident, poetic, strategic
- Highlight keywords in #659157:
  Clarity, Strategy, Identity, Impact

---

### SERVICES SECTION
Grid of services:
- Brand Strategy
- Identity Design
- Web Experience Design
- Creative Direction
- Digital Systems
- Growth & Positioning

Cards:
- Title in #221e22
- Description in #7f6a93
- Hover accent in #659157

---

### CASE STUDIES / WORK
- Editorial list/grid
- Each project:
  - Title
  - Category label (#7f6a93)
  - Short description
- Hover:
  - Image reveal or zoom
  - Accent underline (#9e6240)

---

### PROCESS SECTION
Steps:
1. Discover
2. Define
3. Design
4. Develop
5. Deploy

Design:
- Large faint step numbers
- Clean vertical storytelling flow
- Accent highlights in #659157

---

### INSIGHTS / BLOG
- Minimal article list
- Title + date + category
- Hover highlight in #659157

---

### FINAL CTA
- Full-width closing section
- Statement: “Let’s build something meaningful.”
- CTA button in #9e6240
- Contact + social links

---

## 6. INTERACTIONS

- Smooth scroll behavior
- Fade + translate transitions between sections
- Subtle parallax in hero
- Hover micro-interactions
- Mobile stacked layout
